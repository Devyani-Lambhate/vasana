# Vasana · वासना

**Environmental Intelligence Initiative**

Vasana is an open research project measuring the latent impressions of human action on the natural world. The name comes from Sanskrit — vāsanā (वासना) refers to the subtle impressions left by past actions that persist and shape what follows.

## Projects

| # | Project | Status |

| 01 | AI Datacenter Environmental Impact | Active |
| 02 | Fast Fashion & Textiles | Upcoming |
| 03 | Urban Construction | Upcoming |

## Deploy to GitHub Pages

1. Fork or clone this repository
2. Go to Settings → Pages
3. Under Source, select `main` branch and `/ (root)`
4. Click Save — your site will be live at `https://yourusername.github.io/vasana`

## Local development

No build step required. Open `index.html` directly in a browser, or use a simple local server:

```bash
npx serve .
# or
python3 -m http.server 8000
```
